Trading Backend Rust · Axum

Learns trading strategies from your notes (via an LLM), evaluates them against live forex & derivative-index market data, and trades for you in paper / signals / live mode — with analytics, insights, and backtesting.

Engine v5.0 — Evidence-Based Multi-Timeframe Analysis
✓ 5000-candle MTF (15min · 1H · 4H · Daily)   ✓ Candlestick + Chart Patterns   ✓ Notes + Rules DSL
✓ Macro Trend Filter (blocks counter-trend)   ✓ Trigger Gate (no early entries)   ✓ 2-Candle-Ahead Prediction
✓ Adaptive Conviction (65% bar)   ✓ Session Quality   ✓ Fatigue Detection   ✓ Trade Options (non-contradictory)
✓ News Assessment   ✓ Forward Forecast + Move Duration   ✓ Symbol-Aware Timeframe Recommendations

Accounts & mode

Create an account and switch autonomy: paper / signals / live.

POST   /accounts
GET    /accounts
GET    /accounts/:id
POST   /accounts/:id/mode  {"mode":"paper"}

Strategies (manual)

Author or edit rule-based strategies in the rule DSL.

POST   /accounts/:id/strategies
GET    /accounts/:id/strategies
GET    /strategies/:id
PUT    /strategies/:id
DELETE /strategies/:id

Notes → LLM extraction

Upload trading notes; the LLM extracts an executable strategy.

POST /accounts/:id/notes
POST /notes/:id   (trigger extraction)

Backtesting

Replay historical candles through a strategy's rules.

POST /strategies/:id/backtest
{"symbol":"frxEURUSD","candles":1000}

Signals & trades

Inspect emitted signals and open/closed trades.

GET  /accounts/:id/signals
GET  /accounts/:id/trades
POST /trades/:id/close {"exit_price":1.1}

Analytics & insights

PnL, win rate, per-strategy perf, narrative insights.

GET /accounts/:id/analytics
GET /accounts/:id/insights

Try it

Create an account in paper mode:

curl -X POST http://localhost:8080/accounts \
  -H 'content-type: application/json' \
  -d '{"label":"demo","broker":"oanda","account_ref":"acct1","balance":10000}'
Default mode: paper. Set broker tokens in .env to enable live trading.
